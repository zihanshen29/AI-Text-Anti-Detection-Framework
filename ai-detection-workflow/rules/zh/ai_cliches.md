# 中文 AI 套话规则库 (AI Cliché Vocabulary)

> **由 Layer 0 (Discovery) 在 Step 4 扫描时加载。**
> **Layer 2 (Execution) 不读此文件** —— 执行层只用 `plan.md` 里的 exact before/after。

## 为什么中文规则库以词汇为主，英文规则库以句法为主

中英两种语言的 AI 检测信号分布不同。中文检测器（知网 AIGC、万方、维普、笔灵、PaperPass）主要依赖**固定套语匹配**和**词频特征**，因为：

1. 中文句法更灵活，同一意思可以有非常多的语序安排，LLM 并不会表现出明显的句长均匀性特征；
2. 但中文的学术套话（"综上所述"、"由此可见"）和新兴黑话（"赋能"、"抓手"、"闭环"）都高度固化，LLM 对这些词汇有强烈偏好，因为训练数据里过多；
3. 中文检测器的词向量模型特别训练过 AI 偏爱词典，命中一个就移动几个百分点。

所以这个文件以**词汇 + 短语清单**为主，以句法模板为辅。和 `rules/en/sentence_patterns.md` 的结构不对称是刻意的。

## 阅读格式

每条规则包含：

- **C-NN** —— 规则编号，`discovery.md` 和 `plan.md` 里交叉引用用。
- **词/短语清单** —— 具体匹配串。所有都是字面匹配，不用正则或模糊。
- **密度阈值** —— 多少字内出现几次才算 hit。中文真人写作里偶尔出现这些词是正常的，密度才是信号。
- **为什么被标记** —— 这一类 AI 偏好背后的一句话解释。
- **体裁相关性** —— 学术 / 商业 / 社交 / 新闻 等不同体裁下的相关性。有些词在学术里几乎不出现（放进学术规则不会触发），但商业文案里是主力。
- **修改方向** —— 一般指导，不给 exact rewrite。

## 一、学术套语连接词

### C-01: 万能承上启下套语

- **词表：** 综上所述、由此可见、不难看出、不难发现、值得注意的是、需要指出的是、必须强调、毋庸置疑、无可厚非、从某种程度上讲、从某种意义上讲、在一定程度上、在某种程度上、可以这样说、换言之、换句话说、也就是说、简而言之、一言以蔽之、总的来说、总而言之、概括起来说
- **密度阈值：** 每 2000 字出现 > 3 次即 hit；整篇论文内出现 > 10 次是强信号。
- **为什么被标记：** 这些是 LLM 中文学术输出的"标配过渡词"。训练数据里过量，生成时频繁调用。真人学术写作会变换说法，也会很多时候直接省略不写。
- **强敏感检测器：** 知网 AIGC、万方、笔灵。
- **体裁相关性：** 学术论文高频（★★★），政策公文高频（★★★），商业文案中频（★★），个人博客低频（★），文学作品极低（应保留）。
- **修改方向：** 多数可以直接删除，让下一句直接承接。若必须保留过渡，用更具体的衔接（"这一结果表明..."、"上述分析说明..."），避免万能套语。

### C-02: 开场定位套语

- **词表：** 在当今社会、在当前背景下、在新时代、随着...的发展、随着...的不断深入、随着...的日益普及、在...的推动下、伴随着、在全球化的今天、在信息化时代
- **密度阈值：** 文档或章节开头出现任意一次即 hit（这些一般只会在开头出现一次，出现就是信号）。
- **为什么被标记：** LLM 生成的中文议论文/论文常常以时代背景铺垫开头而不是直接进入论点。GPT 系列特别倾向这种开局。
- **强敏感检测器：** 知网 AIGC、万方、PaperPass。
- **体裁相关性：** 学术论文（★★★ 高危），新闻社论（★★），商业文案（★★★），文学（应保留）。
- **修改方向：** 删除背景套语，让开头直接是具体论点或具体研究对象。一篇关于四足机器人控制的论文，不需要以"随着人工智能的发展"开头。

### C-03: AI 式程度强调

- **词表：** 极其、非常、十分、格外、尤为、尤其、特别是、相当、极为、颇为、很是、甚为
- **密度阈值：** 每 500 字出现 > 3 次即 hit，连续两句都用同类词也算 hit。
- **为什么被标记：** LLM 中文输出会过度使用程度副词填充。真人写作里，具体的量化（"提高 48%"、"超过基线 0.3 个标准差"）会替代抽象强调。
- **强敏感检测器：** 维普、笔灵（词频模型）。
- **体裁相关性：** 学术论文（★★★ 高危，应用具体数据替代），商业文案（★），文学（可保留风格）。
- **修改方向：** 能用数据就用数据；不能用数据就直接去掉副词。"非常重要" → "关键" 或 "影响 X 的主要因素"。

## 二、AI 偏爱抽象名词 / 新兴黑话

### C-04: 互联网/商业黑话

- **词表：** 赋能、抓手、痛点、闭环、生态、壁垒、颗粒度、心智、势能、纬度（常误用）、对标、对齐、打通、下沉、抢占、破圈、出圈、体感、触达、链路、调性、完备、复盘、锚点、底座、底层逻辑、顶层设计、战略高地、第二曲线、飞轮效应
- **密度阈值：** 学术/技术论文里出现任意一次即 hit（正常学术写作不会用这类词）；商业文案里密度 > 1 per 300 字才算 hit。
- **为什么被标记：** 这些词在互联网公司语境里高频，被大量投喂进 LLM 训练语料。生成时容易溢出到本不该用的体裁。学术论文里一旦出现，检测器高置信度判定 AI。
- **强敏感检测器：** 知网 AIGC、笔灵、PaperPass。
- **体裁相关性：** 学术论文（★★★ 严重，几乎是硬信号），技术博客（★★），商业战略文案（★），产品经理文档（可接受）。
- **修改方向：** 替换为具体的技术名词。"给 X 赋能" → "提升 X 的性能"/"扩展 X 的能力"；"抓手" → "切入点"/"方法"/"手段"；"痛点" → "问题"/"困难"/"瓶颈"；"闭环" → "完整流程"/"反馈回路"。

### C-05: "化"字后缀过量

- **词表：** 系统化、结构化、精细化、可视化、模块化、流程化、标准化、规范化、常态化、体系化、数字化、智能化、自动化、科学化、精准化、个性化、定制化、一体化、平台化、生态化、产业化、规模化、长效化、立体化、纵深化
- **密度阈值：** 每 500 字出现 > 3 个不同的 "化" 字词即 hit，连续三句都带"化"字后缀即强信号。
- **为什么被标记：** "化"字是中文 LLM 偏爱的抽象后缀，用来把具体动作抽象成管理学术语。真人写作里会有但不会这么密。
- **强敏感检测器：** 知网 AIGC、万方、笔灵。
- **体裁相关性：** 管理学论文（★，部分合理），技术论文（★★★），政策文件（★，常态），文学（极少，应保留）。
- **修改方向：** 改回具体动作描述。"流程化处理" → "按固定步骤处理"；"可视化呈现" → "用图表展示"；"系统化研究" → "系统研究" 或 "全面研究"。

### C-06: 三字式 / 四字式词组堆叠

- **词表示例：** 全面提升、深度融合、深入探讨、扎实推进、有机结合、有效衔接、稳步推进、持续优化、不断完善、进一步加强、有力保障、切实做到、统筹兼顾、协调推进、通盘考虑、综合施策、精准发力、系统谋划
- **密度阈值：** 任意三句内出现 2 个以上即 hit；整段内出现 > 3 个不同的四字组合是强信号。
- **为什么被标记：** 这是 LLM 中文"政务文风"的核心特征。训练数据包含大量政府报告和新闻稿，模型会在正式语境里自动调用这一模式。学术写作里真人不这么说话。
- **强敏感检测器：** 知网 AIGC、万方、笔灵（这个几乎是中文 AI 检测的头号信号）。
- **体裁相关性：** 学术论文（★★★ 严重），技术文档（★★★ 严重），政策文件（★ 合理），商业提案（★★）。
- **修改方向：** 拆解成具体行为。"全面提升 X 的性能" → "在 A、B、C 三个方面提升 X" 或 "将 X 的 A 指标从 0.3 提升到 0.5"。"深度融合" → 描述具体怎么融合。

### C-07: 大而空的抽象名词

- **词表：** 高度、角度、层面、维度、方面、领域、范畴、视野、格局、站位、高度重视、深度思考、宽度/厚度（抽象用法）、广度和深度
- **密度阈值：** 每 500 字出现 > 4 次即 hit；同一段出现 "从...角度" + "在...层面" + "在...维度" 组合即强信号。
- **为什么被标记：** 这些词是 LLM 用来显得"有思考"的占位符，真人表达会直接说具体是什么角度/层面。
- **强敏感检测器：** 知网 AIGC、笔灵。
- **体裁相关性：** 学术论文（★★），政务文档（★），评论文章（★★），文学（低）。
- **修改方向：** "从 X 的角度来看" → 直接陈述观点，删除框架词；或具体化为 "从 X 的实验数据 / 理论推导 / 应用案例 来看"。

## 三、完美列举与结构化套路

### C-08: 首先其次再次最后

- **词表：** 首先、其次、再次、然后、最后；第一、第二、第三、第四；一是、二是、三是、四是
- **密度阈值：** 同一段落或相邻段落出现完整 3-4 元序列即 hit。单个 "首先" 出现不算。
- **为什么被标记：** 真人学术写作偶尔用枚举，但不会用满整个四元序列。LLM 会把一段论证机械拆成 "首先/其次/再次/最后"。
- **强敏感检测器：** 知网 AIGC、万方、维普。
- **体裁相关性：** 学术论文（★★★ 严重），演讲稿（★，合理），商业报告（★★），文学（★★★ 严重）。
- **修改方向：** 打散成自然过渡。"首先是 A... 其次是 B... 再次是 C..." → "A... B 与 A 相关但不同之处在于... C 则进一步说明..."。

### C-09: 不仅...而且/更... 套句

- **词表：** 不仅...而且、不仅...还、不仅...更、不但...而且、既...又...、一方面...另一方面、既要...又要...、要...也要...
- **密度阈值：** 每 800 字出现 > 2 次即 hit。
- **为什么被标记：** 工整对仗结构是 LLM 修辞偏好。真人写作里会用但不会这么密集，且常会被打破节奏。
- **强敏感检测器：** 知网 AIGC、笔灵。
- **体裁相关性：** 学术论文（★★），议论文（★★），报告（★★），文学（★）。
- **修改方向：** 拆成两个独立句子，或保留一个对仗结构在段落重点处，把其他改为自然陈述。

### C-10: 因果链套句

- **词表：** 之所以...是因为、正是因为...所以、由于...因此、鉴于...故而、之所以...在于
- **密度阈值：** 每 1500 字 > 2 次即 hit。
- **为什么被标记：** LLM 喜欢用显式因果连接填充句子。真人常常用更含蓄的因果（"X，这导致 Y" / "X 让 Y 成为可能"）。
- **强敏感检测器：** 知网 AIGC、万方。
- **体裁相关性：** 学术论文（★★），法律文书（★，合理），新闻（★★），文学（★）。
- **修改方向：** 改为简洁的 "X 导致 Y" 或 "因 X 而 Y"，或直接让因果关系从上下文推导出。

## 四、对偶与修辞套路

### C-11: 空洞排比

- **词表结构：** 连续三个或以上以相同句式开头的短句，例如 "是... / 是... / 是..."、"不是... / 而是... / 更是..."、"需要... / 需要... / 需要..."
- **密度阈值：** 任意段落内出现一次完整三元排比即 hit。
- **为什么被标记：** LLM 把排比当作修辞"加强手段"使用。真人排比少，且通常有具体内容；LLM 排比经常空洞（"是挑战，是机遇，是未来"）。
- **强敏感检测器：** 知网 AIGC、笔灵。
- **体裁相关性：** 学术论文（★★★ 严重），演讲稿（★ 合理），散文（★，视内容）。
- **修改方向：** 把三元排比改为一到两句陈述性的具体表述，砍掉抽象标签。

### C-12: 起承转合套语

- **词表：** 然而、但是、不过、尽管如此、话虽如此、与此同时、与之相对、相较而言、此外、除此之外
- **密度阈值：** 每 800 字 > 3 次同类转折词即 hit；连续段落每段都以转折词开头是强信号。
- **为什么被标记：** LLM 过度使用显式转折标记来营造"有思辨"的错觉。真人写作里转折常常内嵌于句中或依靠上下文推断。
- **强敏感检测器：** 知网 AIGC、万方。
- **体裁相关性：** 学术论文（★★），议论文（★★），报告（★★），文学（★）。
- **修改方向：** 删除一部分显式转折词，依靠语义自然衔接；或换更具体的衔接（"X 的结果出人意料，因为..."）。

### C-13: 程度递进"化"系列

- **词表：** 越来越、日益、日趋、愈发、愈加、更加、进一步
- **密度阈值：** 每 500 字 > 3 次即 hit；同段 "越来越..." 反复出现即强信号。
- **为什么被标记：** LLM 用这类程度递进词填充趋势判断，但很少给出具体时间/数据支撑。
- **强敏感检测器：** 知网 AIGC、维普。
- **体裁相关性：** 学术（★★★），新闻（★★，可接受），社论（★）。
- **修改方向：** 换成具体数据或时间范围。"越来越受关注" → "近五年文献数量增长了 3 倍" 或 "自 2020 年以来"。

## 五、句末语气与收尾套路

### C-14: 结尾抒情套语

- **词表：** ...必将、...势必、...指日可待、...前景广阔、...大有可为、...任重道远、...未来可期
- **密度阈值：** 任意出现即 hit（这类词在学术写作里基本不该出现）。
- **为什么被标记：** 典型"宣传文风"残留，被 LLM 从训练语料里学走并用在不合适的语境。
- **强敏感检测器：** 知网 AIGC、笔灵、PaperPass。
- **体裁相关性：** 学术论文（★★★ 严重，几乎是硬信号），政策文件（★ 常见），新闻（★★），商业文案（★）。
- **修改方向：** 直接删除整句，或改为具体的限定性陈述（"在解决 X 问题上仍有改进空间"）。

### C-15: "这一 X"指代链

- **词表结构：** "这一问题"、"这一挑战"、"这一现象"、"这一趋势"、"这一过程"、"这一机制"
- **密度阈值：** 每 600 字 > 3 次同类"这一 N"结构即 hit。
- **为什么被标记：** 中文 LLM 过度使用"这一 + 名词"作为跨句衔接，而真人写作更常用明确的名词重复或代词。
- **强敏感检测器：** 知网 AIGC、笔灵。
- **体裁相关性：** 学术论文（★★），新闻（★），文学（★）。
- **修改方向：** 把一部分 "这一 X" 换为具体名词或省略。

## 六、新兴 LLM 特有中文模式

### C-16: 过度周全的免责 / 平衡表述

- **词表：** 需要辩证看待、一分为二、既有积极的一面，也有...、不可一概而论、需要具体问题具体分析、因情况而异
- **密度阈值：** 每 2000 字 > 2 次即 hit。
- **为什么被标记：** LLM 在 RLHF 训练后特别容易生成"两面都要看到"的平衡话术，即使上下文不需要。真人论证更倾向于有立场。
- **强敏感检测器：** 笔灵、PaperPass。
- **体裁相关性：** 学术论文（★★★，在论点处不该有），评论（★★），政策（★）。
- **修改方向：** 在作者明确有立场的地方，删除平衡话术。只在真正需要承认反方的位置保留，且紧接着给出回应。

### C-17: 伪总结式反问

- **词表：** 为什么会这样？、问题出在哪里？、答案是...、原因有三：、这是为什么？、何以至此？
- **密度阈值：** 每 3000 字 > 2 次即 hit；连续段落中都用反问开头是强信号。
- **为什么被标记：** LLM 把自问自答当作"制造节奏"的修辞手段。学术写作不会这么说话。
- **强敏感检测器：** 知网 AIGC、笔灵。
- **体裁相关性：** 学术论文（★★★），科普（★★，可接受），演讲（★，合理）。
- **修改方向：** 把反问改为陈述句。"为什么会这样？是因为 X" → "这一现象的原因是 X"。

### C-18: "AI 味"的顿号堆叠

- **词表结构：** 同一句内超过 4 个并列项用顿号隔开。例如 "X、Y、Z、W、V 都是..."
- **密度阈值：** 每 500 字出现 > 2 次长顿号链即 hit。
- **为什么被标记：** LLM 中文输出常用长顿号清单来"显得全面"，但每一项都抽象。真人写作或控制项目数（2-3 个），或明确分点。
- **强敏感检测器：** 知网 AIGC、维普。
- **体裁相关性：** 学术论文（★★），报告（★★），新闻（★）。
- **修改方向：** 拆成分点列表（如果确实有 4+ 个并列项），或合并到 2-3 个概括项。
